# reCloudStream LaMovie Extension

Spanish movies & series from https://la.movie using Vimeo HLS streams.

## Installation
1. Add repo: `https://raw.githubusercontent.com/rheno911/recloudstream-lamovie/master/repo.json`
2. Search any movie → LaMovie appears
3. Play → Vimeo HD 1080p streams

## Features
- Movie ID: 61323 (Una loca película de vampiros)
- Vimeo HLS with proper Referer headers
- Spanish content focus
